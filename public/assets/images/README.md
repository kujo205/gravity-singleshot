# Image Assets

Place your image assets here:

- `ui/` - UI elements and buttons
- `sprites/` - Game sprites and textures
- `backgrounds/` - Background images

## Suggested Image Files

### UI
- `button-normal.png` - Button texture
- `panel.png` - Panel texture
- `star.png` - Star icon

### Sprites
- `ball.png` - Player ball sprite
- `platform.png` - Platform texture
- `obstacle.png` - Obstacle texture
- `checkpoint.png` - Checkpoint texture

### Backgrounds
- `menu-bg.png` - Menu background
- `game-bg.png` - Game background pattern
